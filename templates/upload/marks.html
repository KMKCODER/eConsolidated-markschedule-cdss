CSV Format Required

Your CSV file must have these columns (in any order):

  • learner_name - Full name of the learner (must exist)
  • test1_mark - Test 1 mark (0-40)
  • test2_mark - Test 2 mark (0-60)
  • class_name - Exact name of the learner's class
Example CSV content:
learner_name,test1_mark,test2_mark,class_name
John Smith,35,,Grade 10A
Jane Doe,,52,Grade 10A
Mary Johnson,32,48,Grade 10B
